# Example. Android RSS-client

This repository contains a detailed sample application that implements MVP Architecture using:

 - Kotlin
 - Anko
 - Android Architecture Components
 - RxJava2
 - Dagger2
 - Retrofit2
 - Espresso
 - Robolectric
